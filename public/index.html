How It Works

This website dynamically loads API scraping scripts from the /api folder.

Just add JavaScript files to the api folder with the proper structure, and they will be automatically loaded.

Direct API Access

You can access APIs directly using these routes:

/api/gpt?query=Hello
/api/pinterest?search=cats
/api/youtube?url=https://youtube.com/...
