How It Works

This website dynamically loads API scraping scripts from the /api folder.

Each API is categorized and displayed in a card with its required parameters.

Simply fill in the required fields and click "Fetch Data" to get results.

Direct URL Access

You can access APIs directly using URL parameters:

/gpt?query=Hello
/pinterest?search=cats
/youtube?url=https://youtube.com/watch?v=abc123
